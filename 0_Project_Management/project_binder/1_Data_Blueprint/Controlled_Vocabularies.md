# Controlled Vocabularies

This document provides the official, standardized lists of terms to be used for specific data fields during transcription. Using these exact terms is critical for ensuring the quality and searchability of the final database.

---

## 1. Clemency Decision (`decision_type`)

This is a complete list. Do not add new terms without project leader approval.

- Pardon
- Commutation
- Reprieve
- Restoration
- Denied
- Term Expired
- Other
